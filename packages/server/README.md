# @grimoire-ai/server

Fastify HTTP server for [Grimoire AI](https://grimoireai.quest) — local-first, AI-native requirements management.

This package provides the API server and static file serving for the Grimoire web UI. It is used internally by the CLI's `grimoire ui` command.

## Installation

You probably want the CLI instead:

```bash
npm install -g @grimoire-ai/cli
```

If you need the server library directly:

```bash
npm install @grimoire-ai/server
```

## Documentation

See the [Grimoire AI CLI](https://www.npmjs.com/package/@grimoire-ai/cli) package for full documentation, or visit [grimoireai.quest](https://grimoireai.quest).
